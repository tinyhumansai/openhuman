## Commands

**Install Dependencies**

```console
pnpm install
```

**Start Preview**

```console
pnpm dev
```

**Render a single variant** (produces `out/<CompositionId>.mov` — transparent ProRes 4444)

```console
pnpm render GhostyWave
```

**Render all variants**

```console
pnpm render:all
```

**Upgrade Remotion**

```console
pnpm exec remotion upgrade
```
